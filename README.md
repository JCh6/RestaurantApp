# RestaurantApp
Using Vue.js | Chi | Go | DGraph

## Client Project setup
```
npm install
```
### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```

## Run Server
```
sh start.sh
```

## Run Dgraph Run
```
sh init_dgraph.sh
```
